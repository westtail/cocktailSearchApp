# カクテル検索アプリ 

## 概要
#### カクテルを検索
カクテルの名前や、味、度数、グラスタイプ、ベースのお酒で検索！  
検索したカクテルの作り方も確認することができる  
  
webページ https://www.cocktailsearch.ga
